Funciones Generales de la App

Aca se pueden ver todas las funciones ocupadas para hacer la aplicacion (quizas me olvide alguna :D). Estan separadas por funciones compuestas y funciones basicas con las que estan hechas el resto y casi toda la logica.

Puede ser que exista algun rastro en los argumentos o devoluciones de las funciones un console.log o algo de ello ya que la version preliminar se desarrollo en consola sin manejo del DOM

Funciones Básicas

(Los botones togglean con click para ver las características de las funciones)

pedirTipo()

Pide al usuario que seleccione entre "ingreso" o "gasto"

Retorna: "ingreso" o "gasto"

pedirMonto(tipo)

Pide al usuario un monto numérico mayor o igual a 0

Argumentos: tipo ("ingreso" o "gasto")

Retorna: Número ingresado

pedirDescripcion(tipo)

Pregunta si desea agregar una descripción; si acepta, solicita una

Argumentos: tipo ("ingreso" o "gasto")

Retorna: Texto con la descripción o "No posee una descripción asignada"

pedirEsMensual(tipo)

Confirma si el registro es mensual o no

Argumentos: tipo ("ingreso" o "gasto")

Retorna: true (si es mensual) o false (si no lo es)

pedirFecha(tipo)

Pregunta si se usa la fecha actual o permite ingresar una en formato dd/mm/aaaa

Argumentos: tipo ("ingreso" o "gasto")

Retorna: Fecha en formato "dd/mm/aaaa"

ingresarRegistro()

Llama a las funciones anteriores para pedir datos, genera un ID y guarda el registro

Retorna: No retorna nada, pero agrega el registro a la lista correspondiente

mostrarRegistros()

Muestra todos los ingresos y gastos almacenados en la consola

Retorna: No retorna nada, solo imprime datos en la consola

eliminarRegistro(registro)

Borra un registro de la lista según su ID

Argumentos: Objeto registro a eliminar

Retorna: No retorna nada, solo modifica la lista

opcionesDispAtts(registro)

Muestra los atributos modificables de un registro, excluyendo "id" y "tipo", con su respectivo número de índice para que el usuario pueda seleccionar cuál desea modificar.

Argumentos: Un objeto `registro` que contiene los datos del registro a mostrar.

Retorna: Un arreglo con dos elementos: el primer elemento es un string con los atributos del registro formateados para mostrar al usuario, y el segundo es el número total de atributos disponibles.

buscarRegistro(ID, type)

Busca un registro por su ID dentro de la lista de ingresos o gastos

Argumentos: ID (número) y type ("ingreso" o "gasto")

Retorna: Objeto registro encontrado o undefined si no existe

actualizarTotal()

Recalcula y actualiza los totales de ingresos, gastos y saldo en base a los registros almacenados.

Retorna: No retorna nada, solo actualiza los valores globales en la aplicación.

montoFormateoFloat(montoConFormato)

Convierte un monto formateado en pesos argentinos a un número flotante.

Argumentos: montoConFormato (cadena de texto con formato de moneda)

Retorna: Número flotante con el valor del monto sin formato

montoFormateoPesos(montoSinFormato)

Formatea un número a la notación de moneda en pesos argentinos.

Argumentos: montoSinFormato (número sin formato).

Retorna: Cadena de texto con el monto formateado en pesos argentinos

generarID(tipo)

Genera un nuevo ID único para un registro de ingreso o gasto, incrementando un contador global para cada tipo.

Argumentos: tipo ("ingreso" o "gasto").

Retorna: Número único (ID generado).

asignarAtt(registro, atributo, valor)

Asigna un nuevo valor a un atributo específico de un registro.

Argumentos: registro (objeto del registro), atributo (nombre del atributo), valor (nuevo valor para el atributo).

Retorna: El valor actualizado del atributo en el registro.

esVacio(tipo, mostrarMensaje = true)

Verifica si la lista de ingresos o gastos está vacía y opcionalmente muestra un mensaje en la consola.

Argumentos: tipo ("ingreso" o "gasto"), mostrarMensaje (booleano que indica si se debe mostrar un mensaje en la consola).

Retorna: Booleano (true si está vacía, false si no).

existeId(tipo, valor)

Verifica si existe un registro con un ID específico en la lista de ingresos o gastos.

Argumentos: tipo ("ingreso" o "gasto"), valor (ID a buscar).

Retorna: Booleano (true si existe el ID, false si no).

opcionesDispCat(tipo)

Devuelve una lista de categorías disponibles (ingresos o gastos) con formato para mostrar al usuario.

Argumentos: tipo ("ingreso" o "gasto").

Retorna: String con las categorías disponibles para el tipo especificado.

Funciones compuestas

ingresarRegistro()

Crea un nuevo registro de ingreso o gasto y lo almacena en la lista correspondiente.

Argumentos: Ninguno (solicita los datos al usuario mediante otras funciones).

Retorna: Objeto del registro creado.

mostrarRegistros()

Muestra en la consola todos los registros almacenados, diferenciando entre ingresos y gastos.

Argumentos: Ninguno.

Retorna: Nada (muestra los datos en la consola).

calcularSaldo()

Calcula la diferencia entre ingresos y gastos totales

Retorna: Número con el saldo actual

pedirCategoria(tipo)

Pregunta si desea agregar una categoría y permite elegir una de la lista

Argumentos: tipo ("ingreso" o "gasto").

Retorna: Nombre de la categoría seleccionada o "Varios"

pedirId(tipo)

Pide un ID de registro existente para eliminarlo (Solo para usar las funciones con punteros a registros para asi identificarlos)

Argumentos: tipo ("ingreso" o "gasto")

Retorna: Número del ID ingresado

seleccionarEliminarRegistro()

Permite al usuario seleccionar un registro para eliminar de la lista, mostrando un índice para cada uno y pidiendo confirmación antes de proceder con la eliminación.

Argumentos: Ninguno.

Retorna: Nada. El registro seleccionado será eliminado de la base de datos si el usuario confirma la acción.

editarRegistro()

Permite al usuario modificar los atributos de un registro previamente seleccionado. Muestra las opciones disponibles para editar los campos, excluyendo el campo "id" y "tipo".

Argumentos: Ninguno. El registro a editar se selecciona en un paso previo.

Retorna: Nada. El registro se actualizará con los nuevos valores proporcionados por el usuario.

pedirAttCambiar()

Solicita al usuario que seleccione qué atributo de un registro desea cambiar, mostrándole las opciones disponibles (atributos modificables).

Argumentos: Ninguno.

Retorna: Un valor que corresponde al índice del atributo seleccionado para cambiar.

seleccionarAtt()

Muestra los atributos de un registro y permite al usuario seleccionar cuál desea modificar. Excluye "id" y "tipo" de la selección.

Argumentos: Ninguno.

Retorna: El índice del atributo seleccionado para ser editado.
