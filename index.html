Gestor de Gastos Mensuales Personales

Este proyecto es una aplicación de gestión de gastos personales desarrollada en JavaScript. Permite al usuario registrar ingresos y gastos, clasificarlos en diferentes categorías (fijos, imprevistos y ahorro) y calcular el saldo disponible. La primera versión funciona mediante el uso de `prompt`, `alert` y `console.log`.

Requerimientos Funcionales
(Con expectativas para futuras versiones)

  • Permitir al usuario agregar, modificar y eliminar ingresos y gastos.
  • Clasificar los registros en categorías: fijos, imprevistos y ahorro.
  • Calcular y mostrar el saldo disponible.
  • Alertar si los gastos superan los ingresos.
  • Guardar y cargar datos desde archivos JSON.

Mejoras en Próximas Versiones

En las próximas versiones, se agregarán mejoras para optimizar el rendimiento. Uno de los principales cambios será implementar un seguimiento del número de modificaciones realizadas por el usuario sobre los ingresos y gastos. Esto permitirá el uso de dos archivos JSON separados: uno para los registros activos y otro para mantener los registros históricos inmutables, garantizando la integridad de los datos.

Modo de Uso

Actualmente se interactua con los botones de debajo, se ingresan los datos del usuario SOLO POR PROMPT y se pueden observar las salidas SOLO POR CONSOLA

Pasos:

  1. Abrir la consola con la tecla F12 y ajustar los tamaños del viewport de manera acorde.
  2. Presionar el boton Agregar Registro para ingresar los datos del usuario.
  3. Presionar el boton Ver Registros para ver los datos ingresados.
