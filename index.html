Gestor de Gastos Mensuales Personales

Este proyecto es una aplicación de gestión de gastos personales desarrollada en JavaScript. Permite al usuario registrar ingresos y gastos, clasificarlos en diferentes categorías (fijos, imprevistos y ahorro) y calcular el saldo disponible. La primera versión funciona mediante el uso de `prompt`, `alert` y `console.log`.

Modo de Uso

Actualmente se pueden observar las salidas SOLO POR CONSOLA

Pasos:

1. Agregar un ingreso o gasto: Ejecuta ingresarRegistro();. Se solicitará tipo, monto, categoría, descripción, si es mensual y fecha.

2. Ver todos los registros: Ejecuta mostrarRegistros();. Muestra los ingresos y gastos guardados.

3. Eliminar un registro: Sabiendo el ID y el tipo de registro a eliminar primero usamos esta funcion: const reg = buscarRegistro(id, tipo); Una vez encontrado el registro y apuntado con la variable reg usamos la funcion eliminarRegistro(reg);

4. Ver saldo disponible: Ejecuta console.log(`Saldo actual: ${calcularSaldo()}`);. Calcula ingresos menos gastos.

5. Los datos se pierden al recargar la página. ⚠️ No hay almacenamiento permanente en esta versión.
