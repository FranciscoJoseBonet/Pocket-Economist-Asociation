Tu contador de Bolsillo

Este proyecto es una aplicación de gestión de gastos personales desarrollada en JavaScript. Permite al usuario registrar ingresos y gastos, clasificarlos en diferentes categorías (fijos, imprevistos, ahorro) y calcular el saldo disponible. La primera versión funciona mediante prompt, alert y console.log.

Requerimentos Funcionales
(Con expectativas a futuro)

  • Permitir al usuario agregar, modificar y eliminar ingresos y gastos.
  • Clasificar los registros en categorías: fijos, imprevistos y ahorro.
  • Calcular y mostrar el saldo disponible.
  • Alertar si los gastos superan los ingresos.
  • Guardar y cargar datos desde archivos JSON.

Forma de Uso

Actualmente se inicia con el boton de debajo, se ingresan los datos SOLO POR PROMPT y se pueden observar las salidas SOLO POR CONSOLA, en las próximas versinoes incluiremos el uso del DOM

Cambios esperados a futuro

En las proximas versiones se esperará agregar cambios para mejorar el rendimiento, uno de los principales cambios será llevar el conteo del total de cambios hechos por el usuario sobre los gastos e ingresos para asi poder usar dos archivos JSON con dichos elementos que se mantengan inmodificables para tener en cuenta la inmutablidad de los registros históricos
